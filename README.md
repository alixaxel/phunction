#Phunction

A Small and carefully designed PHP HMVC framework.


##Basic Use
		
	ph()->{class}->{method}
	
ex.

	ph()->Unicode->lcfirst($string);
	

### Classes

* Date
* DB
* DB_SQL
* Disk
* HTML
* HTTP
* Is - variable type checking methods
* Math
* Net
* Net_Akismet
* Net_Google
* Net_PayPal
* Text
* Unicode - unicode-safe versions of php string functions
